# 의존성 주입
* Dependency Injection
* 의존성 존재 : A 객체가 어떤 작업을 수행하기 위해 B 객체를 필요로 함
* 이때 A 객체가 아닌 외부의 C 객체가 B를 생성한 후 이를 전달해서 의존성을 해결하는 방법을 **의존성 주입**이라 함

## 의존성 주입 방식
* 생성자 주입 : 
* setter 주입
* 메서드 주입

## 사용하는 상황
* 실행할 때 마다 의존 대상이 매번 달라지는 일시적인 의존이 필요한 경우 &rarr; 메서드 주입
* 동일한 의존이 필요한 경우 &rarr; 생성자 주입, setter 주입
